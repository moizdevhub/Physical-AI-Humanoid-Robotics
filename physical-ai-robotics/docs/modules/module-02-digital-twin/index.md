# Module 2: Digital Twin

## Overview

Create virtual replicas of humanoid robots using Gazebo and NVIDIA Isaac Sim for safe testing and development.

## Topics Covered

- Simulation environment setup (Gazebo)
- URDF/SDF robot modeling
- Physics simulation
- Sensor simulation
- NVIDIA Isaac Sim integration

## Prerequisites

- Module 1 completion
- ROS 2 Humble installed
- Gazebo Classic or Gazebo Fortress
